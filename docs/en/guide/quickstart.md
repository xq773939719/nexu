# One-Minute Quick Start

## Step 1: Download and Install

Download the [nexu client](https://nexu.io/download), install it, and double-click to open.

![nexu home screen](/assets/nexu-home.webp)

## Step 2: Choose Your Mode

Open **Settings** from the left sidebar and choose an integration mode:

- **Sign in** — select **Nexu Official**, click **Sign in to Nexu**, and high-quality models are ready to use immediately
- **BYOK mode** — select a provider such as Anthropic or OpenAI, paste your API key, then click **Save**

![Choose a model integration mode](/assets/nexu-settings-open.webp)

![Nexu Official sign-in](/assets/nexu-models-official.webp)

## Step 3: Configure a Channel

Pick one channel and complete the setup:

- [Feishu](/guide/channels/feishu) — only needs App ID + App Secret
- [Slack](/guide/channels/slack) — one-click app creation via manifest
- [Discord](/guide/channels/discord) — requires Application ID + Bot Token

Once configured, click **Chat** to start talking to your Agent.
