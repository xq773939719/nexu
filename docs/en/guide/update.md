# Update Guide

## Option 1: Update Inside the Client

1. Open nexu
2. Click **Nexu** in the top menu bar
3. Select **Check for Updates**
4. Follow the prompt if a new version is available

> This is the easiest option and the recommended one.

![Check for Updates](/assets/nexu-check-for-updates.webp)

---

## Option 2: Download the Latest Installer Manually

You can download the latest version from either of these locations:

- **Website:** [https://nexu.io/download](https://nexu.io/download)
- **GitHub Releases:** [https://github.com/nexu-io/nexu/releases](https://github.com/nexu-io/nexu/releases)
