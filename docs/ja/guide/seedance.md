# Seedance 2.0 動画生成

Seedance 2.0 の動画生成を試したいですか？以下の手順で設定を完了すれば、`nexu` との会話からそのまま動画生成を始められます。

## 事前準備

始める前に、次の条件を満たしていることを確認してください。

- nexu クライアントをインストール済みである
- nexu アカウントで正常にサインインできる
- `nexu` とやり取りするための IM チャンネルを 1 つ用意している

まだクライアントをインストールしていない場合は、まず [nexu 公式サイト](https://nexu.io/) または [GitHub リポジトリ](https://github.com/nexu-io/nexu) を確認し、[ダウンロードページ](https://nexu.io/download) から最新版を入手してください。

## ステップ 1: nexu にログインする

クライアントを開き、**nexu アカウントでログイン** を選択します。

ログインに成功すると nexu ワークスペースに入り、続けて設定を進められます。

## ステップ 2: Seedance 2.0 体験 Key を申請する

クライアントのホーム画面で **Seedance 2.0** のキャンペーンバナーを見つけてクリックし、案内に従って申請を進めます。

![ホーム画面の Seedance 2.0 バナー](/assets/seedance/home-banner.webp)

次の作業が必要です。

1. GitHub で `nexu` に Star を付ける
2. グループ参加後、固定メッセージを開き、フォームのリンクから必要情報を記入する
3. 審査結果を待つ

審査に通ると、Seedance 2.0 の体験 Key がフォームに記入したメールアドレスに送信されます。

![体験 Key 申請: GitHub で Star を付ける](/assets/seedance/apply-key-step1-star.webp)

Star を付けたら、ポップアップ内のボタンをクリックしてグループに参加します。

![体験 Key 申請: ボタンをクリックしてグループに参加](/assets/seedance/apply-key-step2-join-group.webp)

グループに入った後は、固定メッセージを開くとフォームのリンクを確認できます。

![体験 Key 申請: 固定メッセージからフォームリンクを開く](/assets/seedance/apply-key-step2-pinned-message.webp)

## ステップ 3: 先に IM チャンネルを設定する

Key が届く前に、先に IM チャンネルを 1 つ設定しておくことをおすすめします。そうしておけば、Key を受け取ったらすぐに `nexu` に送って有効化できます。

普段使っているチャンネルを選び、画面の案内に従って設定してください。詳しくは [チャンネル設定](/ja/guide/channels) を参照してください。

## ステップ 4: Key を `nexu` に送る

メールが届いたら、Seedance 2.0 の体験 Key をコピーし、先ほど設定した IM 会話画面を開いて `nexu` に送信します。

有効化に成功すれば、その会話内で Seedance 2.0 を使った動画生成を始められます。

## ステップ 5: 最初の動画を生成する

有効化が完了したら、`nexu` に動画生成の指示を送るだけです。

例:

> Seedance 2.0 を使って、スポンジ・ボブとパトリックがキスする動画を生成して。

![Seedance 2.0 の動画生成タスク](/assets/seedance/generate-video.webp)

より安定した結果を得たい場合は、プロンプトをできるだけ具体的に書くのがおすすめです。特に次の要素を含めると効果的です。

- キャラクター
- 動作
- シーン
- スタイル
- カメラ表現
- 秒数

例:

> Seedance 2.0 を使って、スポンジ・ボブとパトリックが海辺でキスする動画を生成して。カートゥーン風、表情は大げさ、カメラはゆっくり寄る、長さは 5 秒。

## プロンプト作成のコツ

動画生成では、プロンプトが具体的であるほど Seedance 2.0 が意図を理解しやすくなります。次の順番で書くと整理しやすいです。

1. 誰が主役なのかを書く
2. 何をしているのかを書く
3. 場所や環境を補足する
4. 全体のスタイルを指定する
5. カメラワークやショット表現を加える
6. 希望する長さを書く

短い依頼を、より具体的なプロンプトに膨らませるのもおすすめです。たとえば「キスする動画を生成して」ではなく、「夕日の海辺で 2 人のカートゥーンキャラクターがキスし、表情は大げさで、カメラはゆっくり寄り、長さは 5 秒」と書くと、期待に近い結果になりやすくなります。

## よくある質問

**Q: フォーム送信後、どれくらいで Key が届きますか？**

スタッフが内容を確認します。通常は 2 時間前後で、承認されるとフォームに記入したメールアドレスへ Key が送られます。

**Q: Key が届かない場合はどうすればいいですか？**

まず迷惑メールフォルダを確認してください。それでも見つからない場合は、グループ内でスタッフに確認してください。

**Q: IM チャンネルの設定は先に必要ですか？**

先に設定しておくことをおすすめします。利用開始には Key を `nexu` に送る必要があるため、あらかじめ IM チャンネルを用意しておくとスムーズです。

**Q: なぜプロンプトを詳しく書いたほうがよいのですか？**

より詳しいプロンプトのほうが、結果が安定しやすく、期待に近い動画になりやすいためです。

**Q: 公式の体験 Key は何回使えますか？**

公式の体験 Key は Seedance 2.0 を試すためのもので、通常は動画生成 2 回分のみ利用できます。
