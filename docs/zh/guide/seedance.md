# Seedance 2.0 视频生成

想体验 Seedance 2.0 视频生成？按照下面的步骤完成配置后，即可在 `nexu` 中直接发起生成。

## 准备工作

开始前，请先确认你已经具备以下条件：

- 已安装 nexu 客户端
- 可以正常登录 nexu 账号
- 已准备一个可用的 IM 渠道，用于和 `nexu` 对话

如果你还没有安装客户端，可以先访问 [nexu 官网](https://nexu.io/) 或 [GitHub 仓库](https://github.com/nexu-io/nexu) 了解产品，再从 [下载页面](https://nexu.io/download) 安装最新版客户端。

## 第一步：登录 nexu

打开客户端后，选择 **nexu 账号登录**。

登录成功后，你会进入 nexu 工作台，并可以继续完成后续配置。

## 第二步：申请 Seedance 2.0 体验 Key

在客户端首页找到 **Seedance 2.0** 活动 Banner，点击后按页面提示完成申请流程。

![首页 Seedance 2.0 活动 Banner](/assets/seedance/home-banner.webp)

需要完成以下操作：

1. 在 GitHub 为 `nexu` 点 Star
2. 进群后查看置顶消息，打开问卷链接并填写信息
3. 等待工作人员审核

审核通过后，Seedance 2.0 体验 Key 会发送到你在问卷中填写的邮箱。

![申请体验 Key：前往 GitHub Star](/assets/seedance/apply-key-step1-star.webp)

完成 Star 后，点击弹窗中的按钮加入群聊。

![申请体验 Key：点击按钮加入群聊](/assets/seedance/apply-key-step2-join-group.webp)

进入群聊后，查看群置顶消息，即可找到问卷链接并继续填写信息。

![申请体验 Key：进群后查看置顶消息获取问卷链接](/assets/seedance/apply-key-step2-pinned-message.webp)

## 第三步：先配置一个 IM 渠道

在拿到 Key 之前，建议先完成一个 IM 渠道配置。这样你收到 Key 后，就可以直接把它发送给 `nexu` 进行激活。

选择你常用的渠道，按照页面提示完成配置即可。详细配置说明可参考 [渠道配置](/zh/guide/channels)。

## 第四步：将 Key 发送给 `nexu`

收到邮件后，复制你的 Seedance 2.0 体验 Key，进入刚刚配置好的 IM 对话窗口，将 Key 发送给 `nexu`。

如果激活成功，你就可以开始在对话中调用 Seedance 2.0 生成视频。

## 第五步：生成第一个视频

激活完成后，直接向 `nexu` 发送视频生成指令即可。

例如：

> 使用 Seedance 2.0 模型生成一个海绵宝宝和派大星接吻的视频。

![Seedance 2.0 视频生成任务](/assets/seedance/generate-video.webp)

如果你希望结果更稳定、更接近预期，建议把提示词写完整一些，尽量包含这些信息：

- 角色
- 动作
- 场景
- 风格
- 镜头语言
- 时长要求

例如：

> 使用 Seedance 2.0 生成一个海绵宝宝和派大星在海边接吻的视频，卡通风格，表情夸张，镜头缓慢推进，时长 5 秒。

## 提示词编写建议

同样是“生成视频”，提示词越具体，Seedance 2.0 越容易理解你想要的效果。推荐按下面的顺序描述：

1. 先说清主角是谁
2. 再说明主角正在做什么
3. 补充发生地点或环境
4. 指定整体风格
5. 加上镜头运动或景别
6. 说明期望时长

你也可以把一句简单需求逐步扩展成完整提示词。例如把“生成一个亲吻视频”扩展成“两个卡通角色在海边接吻，夕阳光线，表情夸张，镜头缓慢推进，时长 5 秒”。

## 常见问题

**Q: 提交问卷后，多久能收到 Key？**

提交后，工作人员会进行审核，通常大约需要 2 小时左右。审核通过后，Key 会发送到你在问卷中填写的邮箱。

**Q: 没有收到 Key 怎么办？**

请先检查邮箱垃圾箱。如果仍未收到，可以在飞书群内联系工作人员协助确认。

**Q: 必须先配置 IM 吗？**

建议先配置。Key 需要发送给 `nexu` 才能开始使用，因此先完成 IM 配置会更顺畅。

**Q: 为什么建议把提示词写详细一些？**

更完整的提示词通常会带来更稳定、更接近预期的视频效果。

**Q: 官方发放的体验 Key 可以使用多久？**

官方发放的体验 Key 仅用于体验 Seedance 2.0，通常只包含 2 次视频生成机会。
